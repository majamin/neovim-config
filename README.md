# README

These are configuration files for [Neovim](https://neovim.io/).
`HELP.md` has some useful info if you need help.

# NOTES

- Edit `lua/user/init.lua` to start
- To dig deeper, edit:
    - Options: `lua/user/options.lua`
    - Non-plugin maps: `lua/user/maps.lua`
    - Plugin options in `lua/plugins/*`

# CHANGES
